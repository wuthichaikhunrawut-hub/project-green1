สร้างคำขอรับรองสำนักงานสีเขียว

กรอกข้อมูลและขอบเขตการสำรวจเพื่อส่งเรื่องให้ผู้ตรวจประเมิน

เลือกปีที่ต้องการทำเรื่องขอรับการตรวจประเมิน
หลังจากส่งคำขอรับรองแล้ว สถานะของคำขอจะเป็น "รอการตรวจประเมิน" (PENDING) ผู้ประเมินจะสามารถเข้ามาดูและเริ่มการประเมินได้
คำแนะนำ
  • ตรวจสอบปีที่ต้องการประเมินให้ถูกต้องก่อนกดส่งคำขอ
  • ในช่องหมายเหตุ ควรระบุรายละเอียดองค์กรเบื้องต้นเพื่อความสะดวกในการจัดสรรผู้ตรวจ
  • องค์กรสามารถมีคำขอที่มีสถานะ รอการตรวจประเมิน ได้เพียง 1 รายการต่อปีเท่านั้น
  • หากส่งคำขอแล้ว พบข้อมูลผิดพลาด สามารถแจ้งแอดมินหรือรอผู้ประเมินตีกลับให้แก้ไขได้
